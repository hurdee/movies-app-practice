# Консольний каталог фільмів та серіалів

Це просунутий консольний проєкт для керування власною медіатекою. Програма дозволяє не просто зберігати список, а інтегруватися з сервісом TMDB, переглядати постери в ASCII-арті та експортувати дані.

## Основний функціонал:
- **Керування:** Додавання (вручну або через API), видалення та редагування записів.
- **Підтримка типів:** Повноцінна робота з фільмами та серіалами.
- **Пошук та фільтрація:** Пошук за назвою, фільтрація за жанром, статусом перегляду або типом контенту.
- **API TMDB:** Можливість автоматичного додавання даних (назва, рік, опис, постер, рейтинг) прямо з бази даних The Movie Database.
- **Візуалізація:** Генерація ASCII-арт постерів з URL-зображень.
- **Корисні інструменти:**
    - Сортування за рейтингом.
    - Експорт даних у файл `movies.csv` (розділювач `;`).
    - Перевірка посилань на трейлери (автоматичний тест на "биті" посилання).
    - Статистика переглядів та середньої оцінки.

Для швидкого перегляду роботи програми без ручного додавання фільмів:
