# Консольний каталог фільмів

Це навчальний проєкт, розроблений в межах практичної роботи.

## Функціонал:
- Зберігання даних у форматі JSON.
- Додавання нового фільму (назва, жанр, рік, статус).
- Фільтрація за жанром або статусом.
- Оцінка фільмів.
- Обробка некоректного введення користувача.

## Як запустити:
1. Переконайтеся, що у вас встановлено Python.
2. Клонуйте репозиторій: `git clone <url_вашого_репозиторію>`
3. Запустіть програму: `python movies_app.py`
